iRpgFormatter Plug-in

Hi, this is the home of the iRpgFormatter plug-in for IBM Rational Developer for i.

 Version: 1.0.0.b009

Introduction

iRpgFormatter is an open source plug-in for IBM Rational Developer for i. It formats free-form RPG IV source code to improve readability and consistency.

iRpgFormatter features are driven from our ideas and needs, but everybody is encouraged to contribute suggestions and manpower to improve the power of iRpgFormatter.

The iRpgFormatter Project Team


Tools/400, Thomas Raddatz

Features

  • Formats free-form RPG IV source code.
  • Configurable keyword casing (upper case, lower case, capitalized).
  • Configurable indentation of nested code blocks.
  • Formats procedure and keyword parameters.
  • Format source members directly from the Remote Systems view.
  • Format source members directly in an Lpex editor view.
  • Optionally executes the IBM formatter.
  • Optionally format source on save.
